# 图速 ImageBatch

批量图片处理工具，8大功能一站式解决日常图片处理需求。

## 功能

- **压缩** - 批量压缩图片，支持调整质量和目标格式
- **修改尺寸** - 预设小红书/公众号等常用尺寸，支持中心裁剪/缩放填充/缩放到适合三种模式
- **调整DPI** - 修改DPI元数据，不改变像素尺寸，打印必备
- **添加水印** - 文字水印（自定义内容/大小/透明度/位置/平铺）+ 图片水印
- **格式转换** - PNG/JPG/WebP/BMP/HEIC 互转
- **添加边框** - 自定义边框宽度/颜色/圆角/阴影
- **九宫格切图** - 将图片切割为3x3九宫格，可调间隙和背景色
- **去除水印** - 框选水印区域，基于OpenCV inpaint算法智能去除

## 特性

- 拖放添加图片，操作直观
- 处理前后预览对比
- 批量处理，显示处理前后文件大小和尺寸变化
- 支持 JPG/PNG/BMP/WebP/HEIC 格式

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python 启动器.py
```

## 系统要求

- Windows 10+
- Python 3.8+
- 依赖: Pillow, pillow-heif, opencv-python, numpy
